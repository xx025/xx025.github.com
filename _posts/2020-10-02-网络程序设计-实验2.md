1.编写一个程序，求无穷级数的和sum=1+1/2+1/3+…1/n ，直到最后一项的精度小于0.000001。'''
```python
def sum_1_n(n, accuracy):
    sum = 0
    for i in range(1, n + 1):
        e = 1.0 / i
        if e >= accuracy:
            sum += e
        else:
            break
    return sum


if __name__ == '__main__':
    print('''求无穷级数的和sum=1+1/2+1/3+…1/n ''')
    n = int(input("请输入n:"))
    print("sum=1+1/2+1/3+…1/n={}".format(sum_1_n(n, 0.000001)))
```
2.给定字符串“The quick brown box ...?”分离其中所有单词，并统计一下每个单词出现次数，按出现次数降序输出

```python
import re

def sum_1_n(s):
    word_dict = dict()
    count_word = dict()
    sum = re.split('[ .?]', s)
    for i in sum:
        if i != '':
            word_dict.setdefault(i, []).append(i)
    for i in word_dict:
        count = 0
        for j in word_dict[i]:
            count += 1
        count_word[i] = count

    return sorted(count_word.items(), key=lambda x: x[1], reverse=True)


if __name__ == '__main__':
    print('''给定字符串
    “The quick brown box jumped over the lazy dog. An apple a day keeps the doctor away. Can a fox and a dog be friends?”
分离其中所有单词，并统计一下每个单词出现次数，按出现次数降序输出: ''', end='\n\n')
    s = "The quick brown box jumped over the lazy dog. An apple a day keeps the doctor away. Can a fox and a dog be friends?"
    print("单词出现的次数:")
    word_count = sum_1_n(s)
    print("{:<8}\t{}".format("单词", "次数"))
    for i in word_count:
        print("{:<8}\t{}".format(i[0], i[1]))

```
3.随机播放一个钢琴的音高（范围：1-7），从控制台输入一个音高值，判断是否正确，并给出提示
```python
import random


def play_sound():
    return random.randint(1, 7)


def play():
    answer = play_sound()
    n = int(input("输入一个音高值(1-7):"))
    if n == answer:
        s = "你输入的正确"
    else:
        s = "你输入的不正确"
    print("\t{}".format(s), end="\n\n")
    play()


if __name__ == '__main__':
    play()
```
